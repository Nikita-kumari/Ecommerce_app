# Ecommerce_app

This is an ecommerce app that sells shoes. The user can browse the app for shoes on sale, and add items to the cart. This app has 3 pages:

Intro page

Shop page

Cart Page

The code is structure cleanly and has been commented in detail so that you don't get lost in the code but if you need further help, you can always reach out!
